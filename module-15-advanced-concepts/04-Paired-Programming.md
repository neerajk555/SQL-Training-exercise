# Paired Programming — Advanced Concepts

## Challenge: Multi-Tenant App with JSON Config — 45 min

**Scenario:** SaaS app where each tenant has custom settings stored in JSON.

**Tasks:**
1. Partner A: Design JSON schema for tenant config
2. Partner B: Implement full-text search across tenant data
3. Together: Build recursive menu system
4. Together: Query tenant-specific settings efficiently

**Discuss:** JSON vs normalized tables, full-text performance, recursive query optimization.
