# Independent Practice — Procedures & Functions

## Exercise 1: Discount Calculator (Easy) — 20 min
Create function to calculate discounted price based on customer tier (bronze=5%, silver=10%, gold=15%).

**Requirements:** Function accepts price and tier, returns discounted price.

---

## Exercise 2: Inventory Management (Medium) — 30 min
Create procedures for: add_stock, remove_stock, check_low_stock. Include validation and error messages.

---

## Exercise 3: Report Generator (Hard) — 40 min
Create procedure that generates sales summary: total sales, top products, revenue by category. Use multiple SELECT statements and temp tables.
